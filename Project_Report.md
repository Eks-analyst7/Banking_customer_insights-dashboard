# Banking Customer Insights Dashboard

## Introduction

The Banking Customer Insights Dashboard is a business intelligence project developed using Power BI to analyze customer financial behavior, banking relationships, deposits, loans, and account performance.

The purpose of this project is to transform banking data into meaningful insights through interactive dashboards and visual analytics. The dashboard helps stakeholders monitor financial performance, understand customer trends, and support data-driven decision-making.

---

## Problem Statement

Banks generate large volumes of customer transaction and account data daily. Analyzing this information manually can be time-consuming and inefficient.

The objective of this project is to create an interactive dashboard that provides a consolidated view of banking operations and customer financial activities, enabling quick analysis and informed decision-making.

---

## Objectives

* Analyze customer deposits and loan activities.
* Monitor key banking performance indicators.
* Identify trends across customer demographics.
* Compare financial performance across customer segments.
* Support business decision-making through interactive reporting.

## Dashboard Overview

The dashboard consists of multiple pages designed to provide insights into different aspects of banking operations.

### Home Dashboard

Provides a summary of major banking KPIs including customer counts, loans, deposits, savings accounts, and checking accounts.

### Loan Analysis Dashboard

Focuses on:

* Total Loan Amount
* Loan Distribution by Occupation
* Loan Distribution by Nationality
* Income Band Analysis
* Banking Relationship Analysis

### Deposit Analysis Dashboard

Focuses on:

* Total Deposit Amount
* Savings Account Analysis
* Checking Account Analysis
* Deposit Distribution by Occupation
* Deposit Distribution by Nationality

### Summary Dashboard

Provides a consolidated overview of key financial metrics and business performance indicators.
