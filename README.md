# Banking Customer Insights Dashboard

## Project Overview

This Power BI project focuses on analyzing banking customer behavior, deposits, loans, and account performance through an interactive dashboard.

The dashboard was designed to provide a comprehensive overview of customer financial activities and support data-driven decision-making within the banking sector.

Using Power BI, I transformed raw banking data into meaningful visualizations that help identify trends, customer segments, and financial performance indicators.

---

## Tools Used

- Power BI
- Microsoft Excel
- Data Modeling
- DAX
- Data Visualization

---

## Dashboard Pages

### Home Dashboard

Provides a high-level overview of banking operations, including:

- Total Clients
- Total Loans
- Total Deposits
- Savings Accounts
- Checking Accounts
- Business Lending

---

### Loan Analysis

Analyzes loan-related metrics including:

- Total Loan Amount
- Bank Loan Distribution
- Business Lending Performance
- Credit Card Balance
- Loan Distribution by Occupation
- Loan Distribution by Nationality
- Income Band Analysis

---

### Deposit Analysis

Provides insights into:

- Total Deposits
- Savings Account Amount
- Checking Account Amount
- Deposit Distribution by Banking Relationship
- Deposit Distribution by Occupation
- Deposit Distribution by Nationality
- Deposit Analysis by Income Band

---

### Summary Dashboard

Combines key financial indicators into a single executive view for quick business analysis.
